# nvim
Nvim config

How to set up:

Install neovim

Install Node

Install C / GCC / Zig / Clang
Windows: https://www.mingw-w64.org/downloads/

Install ripgrep
https://github.com/BurntSushi/ripgrep

Install Language Servers:
npm i -g sql-language-server
npm i -g typescript
npm i -g typescript-language-server
npm i -g pyright

Useful commands:

```
:Mason
:TSInstall
```
